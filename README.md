# 🛍️ VibeShop - Nền tảng E-commerce Cao Cấp

![VibeShop Banner](https://images.unsplash.com/photo-1441986300917-64674bd600d8?q=80&w=2070&auto=format&fit=crop)

VibeShop là một ứng dụng thương mại điện tử hiện đại, được xây dựng với mục tiêu mang lại trải nghiệm mua sắm tinh tế, nhanh chóng và bảo mật. Dự án là sự kết hợp giữa thiết kế Premium và công nghệ web tiên tiến nhất.

## 🚀 Tính Năng Nổi Bật

- 💎 **Giao diện Premium:** Thiết kế sang trọng với animation mượt mà (Framer Motion).
- 🛒 **Giỏ hàng Thông minh:** Quản lý sản phẩm, cập nhật số lượng và tính tiền thời gian thực.
- 🔐 **Bảo mật Supabase:** Hệ thống đăng nhập, đăng ký và bảo vệ dữ liệu người dùng.
- 📜 **Lịch sử Đơn hàng:** Theo dõi tình trạng và xem lại các giao dịch đã thực hiện.
- ⚡ **Hiệu suất Tối ưu:** Áp dụng Lazy Loading, Code Splitting và SEO chuẩn Google.
- 📱 **Responsive:** Hiển thị hoàn hảo trên cả Desktop và Mobile.

## 🛠️ Công Nghệ Sử Dụng

- **Frontend:** React JS, Vite, Tailwind CSS.
- **State Management:** Zustand.
- **Data Fetching:** TanStack Query (React Query).
- **Backend & Auth:** Supabase.
- **Animations:** Framer Motion, Lucide React icons.
- **Form Handling:** React Hook Form & Yup validation.

## 📦 Hướng Dẫn Cài Đặt

1. **Clone dự án:**
   ```bash
   git clone https://github.com/levanquoc/vibeshop.git
   cd vibeshop
   ```

2. **Cài đặt thư viện:**
   ```bash
   npm install
   ```

3. **Cấu hình biến môi trường:**
   Tạo file `.env` tại thư mục gốc và điền thông tin Supabase của bạn:
   ```env
   VITE_SUPABASE_URL=YOUR_SUPABASE_URL
   VITE_SUPABASE_ANON_KEY=YOUR_SUPABASE_ANON_KEY
   ```

4. **Chạy dự án:**
   ```bash
   npm run dev
   ```

## 🌍 Deployment

Dự án đã được cấu hình sẵn sàng để triển khai lên **Vercel**. Chỉ cần kết nối repository này với Vercel và bạn sẽ có một link website chính thức ngay lập tức.
